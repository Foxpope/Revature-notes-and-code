Intro to CSS

CSS can be included in 3 different ways:

  1. inline styling eith the attribute
  2. internal styling using the style tag in the head of the html doc
  3. external style using the link element in the head of the html doc

CSS rules are defined by key/value pairs, separated by a colon { : }
